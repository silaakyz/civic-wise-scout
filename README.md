Bu proje, MVC mimarisi, REST API tasarımı ve sunucu taraflı CRUD işlemlerini uygulamak için yapılmıştır. Öğrencilerin gerçekçi bir iş problemi üzerinden:

- Sunucu taraflı yazılım geliştirme,
- MVC mimarisini doğru ve tutarlı biçimde uygulama,
- REST prensiplerine uygun API tasarlama,
- Veri modeli, iş mantığı ve uç noktaları ayrıştırma,
- Okunabilir, sürdürülebilir ve ölçeklenebilir kod yazma becerilerini öğrenmesi amaçlanmıştır.

Yes, you can!

To connect a domain, navigate to Project > Settings > Domains and click Connect Domain.

Read more here: [Setting up a custom domain](https://docs.lovable.dev/features/custom-domain#custom-domain)
